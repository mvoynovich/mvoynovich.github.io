    • Lab Guidelines: (Work this lab in a lab4 branch)
      • Create five items/entries, each linking to a news or blog posting, or any web content you find interesting (from five different sources). Alternatively, you can make it up if you are so inclined.
      • You may use the same items in both feeds.
      • Post the two files onto a "lab 4" page of your "iit" website.
      • The feeds must validate.
      • In your review of the formats - take note of the layout of Channels and Items, etc.
      • Note: View the feed in your browser after posting it to your website: it wont be pretty. Some browsers have built in RSS and Atom readers, but the feed must normally be viewed through a reader (separate app) to be properly rendered.
      • However, since XHTML is XML, we can add style to our XML files, by adding the following underneath the XML prologue: <?xml-stylesheet type="text/css" href="rss.css"?>
        and then adding style to our css using selectors as normal.
      • For this lab, you do NOT need to style your XML - if it looks like plain XML in your browser - this is ok.
      • Add a README.md file to your lab4 directory discussing this lab. Make sure to include a link to your home page in the readme file.
      • When you are finished with lab04 development - Commit your changes to the GitHub branch, issue a PR and merge
      • THEN - Start your Web Servers, SSH into them, and pull the changes from your repo into the root of your class websites (iit folders)
      • ON LMS - submit a zip of your local repo
